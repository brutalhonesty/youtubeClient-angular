Youtube Client Angular
======================

Version
-------
0.0.1

Author

Description
------------
Youtube Client using AngularJS for [MediaCenterJS](https://github.com/jansmolders86/mediacenterjs)

Installation
-------------
```bash
npm install && mv configuration/config.template configuration/config.ini && vim config.ini
```
